# 주제

- 파킨슨병 보행동결 감소를 위한 보행 지원 스마트 글래스 연동 앱

# 배경

파킨슨병의 대표적인 증상 중 하나인 보행동결은 길을 걷다가 다리가 움직이지 않는 증상이다. 기존의 재활 방식은 바닥의 줄무늬를 따라 걷는 것으로 레이저 지팡이와 신발을 보조 도구로 활용한다. 허나 줄무늬 단서가 고정된 형태로 제공되기 때문에 환자의 보행 리듬과 속도 변화에 대응이 불가능하다. 이를 해결하기 위해 스마트 글래스를 활용하여 보행 유도선을 보여주고 개인 맞춤화가 가능하게 만들었다.

# 주요 기능

- 스마트 글래스와 휴대폰 간에 블루투스 기반 통신 기능
- 환자의 키에 따른 유도선 조절 기능
- 유도선의 간격, 높이, 길이, 색상, 속도 조절 기능
- 사용자의 실시간 속도를 감지하여 유도선의 속도를 조절 기능

# 기술

`BluetoothManager`

스마트 글래스와 휴대폰간 블루투스로 연결하여 휴대폰에서 개인 맞춤 설정한 값을 스마트 글래스에 전달

`SensorManager`

선형 가속도 센서를 사용하여 사용자의 속도를 감지하여 유도선의 속도 조절

# 구조

`App Architecture`, `MVVM`

# 결과 및 배운 점

- 제13회 K-해커톤 대회 최우수상 (정보통신산업진흥원 원장상)
- K-해커톤을 준비하면서, 어떻게 하면 우리의 아이디어를 심사위원에게 더 설득력 있게 전달할 수 있을까 깊이 고민했다. 결국 핵심은 이 소프트웨어를 환자가 실제로 사용했을 때 얼마나 효과가 있는지를 직접 증명하는 것이라고 판단했다.
 이를 위해 파킨슨병 환자를 대상으로 한 실제 테스트가 필요하여 울산대학교 신경외과 교수님과 파킨슨병 재활치료센터에 직접 연락을 드려 협조를 요청했다. 다행히 두 곳 모두 긍정적인 답변을 주셔서 환자 테스트를 진행할 수 있었다.
 실제 파킨슨병 환자분들을 마주하고 테스트를 진행하면서, 책이나 이론으로는 절대 알 수 없는 수많은 변수와 현실적인 상황을 경험했다. 무엇보다, 내가 만든 앱이 환자분들에게 실제로 도움이 될 수 있다는 사실을 확인했을 때 큰 보람과 성취감을 느꼈다.
